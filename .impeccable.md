# Nuoma WPP - Design Context

## Target Audience
Single technical operator managing WhatsApp + Instagram automation for a business.
Power user who values information density and keyboard-friendly workflows.

## Use Cases
- Monitor campaign execution in real-time
- Manage contacts and conversations across WA+IG
- Configure automation rules and chatbot flows
- Quick-glance dashboard for operational health

## Brand Personality
- Unique aesthetic that makes people ask "what app is this?"
- Operational and efficient but not boring
- Dark theme, but not the typical "AI dashboard" look
- Information-dense without feeling cluttered

## Design Direction: "Cartographic Operations"
Inspired by topographic maps, mission control interfaces, and data cartography.
The system maps your customer relationships like terrain - each contact is a point,
each campaign is a route, each conversation is a signal.

Key visual elements:
- Contour-line inspired borders and decorative elements
- Monospaced data with proportional UI text
- Color coding by function (emerald=WA, amber=IG, blue=system)
- Micro-grid backgrounds for precision feeling
- Status indicators as signal dots (not badges)
