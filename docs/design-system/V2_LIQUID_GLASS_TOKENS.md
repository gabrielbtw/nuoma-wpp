# V2 Liquid Glass Tokens

Status: uso seletivo em 2026-05-04.

## Uso Permitido

Liquid Glass no V2 significa camada flutuante com `shadow-lift`, contorno claro e fundo controlado. Ele deve aparecer apenas em:

- dialogs;
- sheets;
- popovers;
- dropdowns;
- command palette;
- toasts;
- overlays futuros no WhatsApp embed.

## Uso Bloqueado

Nao usar glass em:

- cards de dashboard;
- listas de contatos/conversas;
- tabelas operacionais;
- cards de campanha;
- paineis de formulario fixos.

Essas areas devem usar superficies flat com contour border para preservar leitura densa e operacional.
